Newsletters von RP Photonics

RP Photonics veröffentlicht die folgenden Blogs, für die kostenlose Newsletters (Blog Letters!) angeboten werden:

  • das Photonics Spotlight, ein Blog über Photonik und Lasertechnologie, der mit der RP Photonics Encyclopedia assoziiert ist
  • die RP Photonics Software News mit Informationen zu Simulations- und Designsoftware im Bereich Photonik

Beachten Sie, dass wir unsere Newsletters verwenden, um nützliche Informationen zu verbreiten – nicht einfach Werbebotschaften!

Sie können die Newsletters mit einem Formular auf der englischsprachigen Seite bestellen. Hier ist eine deutsche Version der dortigen Erklärungen:

  • Wenn Sie ein Newsletter-Abo bestellen, werden Sie eine Bestätigungs-Mail an die eingegebene Adresse erhalten. Nur wenn Sie das Abo bestätigen, indem Sie auf einen Link in dieser Mail klicken (“double opt in”), wird Ihr Abo aktiviert. (Wir wissen nämlich erst dann, dass Sie das Abo tatsächlich selbst bestellt haben.)
  • Sie können das Abo jederzeit beenden, indem Sie das Formular wieder abschicken, wobei die entsprechenden Abos nicht angekreuzt sind. In diesem Fall erhalten Sie eine Mail, die die Abbestellung bestätigt.
  • Wir werden Ihre E-Mail-Adresse nur für den Zweck des Newsletter-Versands nutzen. Niemals werden wir Ihre Daten an Dritte weitergeben. Beachten Sie hierzu auch unsere Erklärung zum Datenschutz (den Abschnitt über Newsletters), den wir sehr ernst nehmen.
  • Die Newsletter-Mails werden von der Adresse subscriptions@rp-photonics.com kommen. Es ist ratsam, dass Sie diese Adresse (oder einfach unsere ganze Domain rp-photonics.com) in eine “white list” Ihres E-Mail-Systems aufnehmen, damit diese Mails nicht etwa als Spam verdächtigt und herausgefiltert werden.
