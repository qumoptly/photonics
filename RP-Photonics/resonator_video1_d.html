Eine Einführung in die Software RP Resonator V4

Machen Sie Ihr Browser-Fenster bitte genügend breit, um das Video in ausreichender Größe zu sehen.

Dieses Video zeigt Ihnen innerhalb von ca. 19 Minuten das Folgende:

  • Sie erhalten einen einfachen Start durch eine Reihe von Demo-Dateien, die typische Fälle des Resonatordesigns abdecken und auch Fälle mit Optiken für einen einfachen Durchgang. Sie füllen jeweils einfach nur ein Formular aus, starten die Berechnung und erhalten die Ausgaben in der Form numerischer Daten und grafischer Diagramme.
  • Die Formulare und Berechnungen sind nicht fest programmiert, sondern in Form von Skripten implementiert. Sie können diese Skripte einfach ändern, beispielsweise um zusätzliche Details einem Resonatormodell zuzufügen, weitere Berechnungen durchzuführen oder zusätzliche grafische Diagramme zu erzeugen.
  • Diverse Demo-Skripte werden gezeigt, die die folgenden Fälle behandeln: einen Ringresonator, die Fokussierung eines Laserstrahls mit Linsen, einen linearen Laserresonator, einen Verstärker mit einer variablen Anzahl von Durchgängen durch den Laserkristall und einen linearen Resonator mit automatischer numerischer Optimierung.
  • Andere Fähigkeiten der Software werden nur kurz erwähnt: verwendbare optische Elemente, Berechnung der Effekte von Fehljustage (die auch als Optimierungskriterium verwendet werden können) sowie zusätzliche berechenbare Größen.
  • Die Benutzer dieser Software erhalten bei der Erstellung von Skripten eine Menge Hilfe – sowohl durch den sehr hilfreichen technischen Support als auch durch die Software selbst, die viele nützliche Features aufweist, z. B. Syntax Highlighting, Parameter-Hilfe, das interaktive Hilfesystem zusätzlich zum PDF-Handbuch, automatische Neuformatierung von Code, nützliche Formulare für die Erzeugung von Resonator-Code sowie die Code Snippets Library mit häufig gebrauchten Code-Elementen.
