Daten für aktive Fasern

Die Software RP Fiber Power wird mit Daten für eine Auswahl von aktiven Fasern geliefert. Es handelt sich einerseits um spektroskopische Daten (Wirkungsquerschnitte, Lebensdauern, Do­tierungskonzentration) und andererseits Details der Wellenleiterstruktur (Größe des Faserkerns, Modenfläche, Propagationsverluste, etc.). Mit solchen Daten können Sie sofort beginnen, die Performance von Faserverstärkern und Faserlasern zu simulieren.

Diverse Daten aus öffentlich zugänglichen Quellen

Die folgenden Daten aus öffentlich zugänglichen Quellen werden zur Zeit zusammen mit RP Fiber Power geliefert:

  • “Yb-germanosilicate”: Daten für eine Ytterbium-dotierte Germanosilikatfaser, aufgenommen von Dr. Paschotta. Da die meisten Ytterbium-dotierten Fasern ein ähnliches Glas enthalten, können diese Daten für viele Fälle verwendet werden.
  • “ErYb-phosphate”: Erbium- und Ytterbium-dotiertes Phosphatglas
  • “Er-fluorophosphate L11”: Erbium-dotiertes Fluorophosphatglas
  • “Er-fluorozirconate F88”: Erbium-dotiertes Fluorozirkonatglas
  • “Er-silicate L22”: Erbium-dotiertes Silikatglas

Daten von Faserherstellern

RP Photonics arbeitet zur Zeit daran, eine Zusammenarbeit mit mehreren Herstellern von dotierten Fasern zu etablieren. Dies bedeutet im wesentlichen,

  • dass Faserhersteller Daten für ihre Fasern an RP Photonics schicken,
  • dass RP Photonics diese Daten in ein Format bringt, mit dem sie direkt in der Software RP Fiber Power verwendet werden können,
  • dass die Kunden, die Benutzerlizenzen der Software kaufen (oder bereits besitzen), diese Daten von RP Photonics bekommen.

Dies ist offensichtlich eine große Hilfe für die Benutzer der Software: Sie können die Performance, die von verschiedenen Fasern zu erwarten wäre, einfach simulieren, ohne sich selbst um die Beschaffung der Daten kümmern zu müssen.

Soweit sind die folgenden Daten verfügbar:

CorActive ist ein Anbieter für eine Auswahl aktiver Fasern, einschließlich einmodiger Fasern und Doppelkernfasern, dotiert mit Ytterbium, Erbium, Erbium/Ytterbium, Neodym, Thulium oder Samarium. Momentan sind Daten für diverse Ytterbium-dotierte Fasern verfügbar; andere werden folgen.
iXblue bietet diverse aktive Fasern an, sowohl einmodiger Fasern als auch Doppelkernfasern, dotiert mit Ytterbium, Erbium, Erbium/Ytterbium, Neodym oder Thulium. Momentan sind einige Daten für Ytterbium-dotierte Fasern in Bearbeitung; andere werden folgen.
NKT Photonics bietet eine Vielzahl von mikrostrukturierten Fasern (photonic crystal fibers) an, einschließlich Ytterbium-dotierten Doppelkernfasern und Doppelkern-Faserstäben (rod fibers) mit extrem großen Modenflächen. Zur Zeit bereiten wir Datensätze für solche Fasern auf.
nLight bietet Erbium- und Ytterbium-dotierte Fasern, sowohl einmodige als auch Doppelkernfasern. Momentan sind Daten für diverse Fasern in Bearbeitung.

Kunden, die bereits eine Benutzerlizenz der Software haben, können neue und aktualisierte Daten via Download erhalten.

Beachten Sie: Im Rahmen des technischen Supports können wir Ihnen Dateien mit Daten für beliebige Fasern auch anderer Hersteller erstellen, soweit solche Daten von den Herstellern verfügbar sind. Beispielsweise hat Nufern solche Daten für viele Fasern veröffentlicht.
