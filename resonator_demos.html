Demonstrations for Models Made with RP Resonator

Various example cases illustrate what you can do with RP Resonator:

  • ring resonator: shows how to define a resonator and generate some graphics
  • titanium–sapphire laser: a more sophisticated example
  • resonator with automatic optimization: demonstrates how a resonator can be optimized
  • multipass amplifier: multipass amplifier setup with automatic geometric calculations
  • achromatic lens: automatic design optimization of a achromat doublet lens

(Click on the links to get the details.)

The input scripts for these and some other cases are available to license holders. For an easy start in a new project, you can simply copy the example file of a case which is closest to the case of interest. However, you may also easily generate a new script using the interactive forms.
